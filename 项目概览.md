## 🚀 快速开始

### 方式一：使用脚本 (推荐)
```bash
# 1. 安装
install.bat

# 2. 配置
# 编辑 backend\.env 文件，填入 API 密钥

# 3. 启动
start.bat
```

### 方式二：手动安装
```bash
# 1. 安装前端依赖
cd frontend
npm install

# 2. 安装后端依赖
cd ../backend
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 4. 启动后端 (终端 1)
npm run dev

# 5. 启动前端 (终端 2)
cd ../frontend
npm run dev
```

### 访问应用
```
http://localhost:3000
```
