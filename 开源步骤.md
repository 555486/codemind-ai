### 5. 📤 推送代码到 GitHub

**在命令行中执行:**

```bash
# 1. 添加远程仓库 (替换 YOUR_USERNAME 为您的 GitHub 用户名)
git remote add origin https://github.com/YOUR_USERNAME/codemind-ai.git

# 2. 重命名分支为 main
git branch -M main

# 3. 推送到 GitHub
git push -u origin main
```

**示例:** 如果您的 GitHub 用户名是 `zhangsan`
```bash
git remote add origin https://github.com/zhangsan/codemind-ai.git
git branch -M main
git push -u origin main
```

## 📞 需要帮助?

### 常见问题

**Q: 推送失败怎么办？**
A: 检查网络连接，确认 GitHub 账号已登录

**Q: 如何修改已提交的内容？**
A: 修改后使用 `git add .` 和 `git commit --amend`，然后 `git push -f`

**Q: 如何删除已提交的文件？**
A: 使用 `git rm --cached <filename>` 然后重新 commit
