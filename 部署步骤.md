##  方案 A: 只部署前端 (推荐)

### 步骤 1: 访问 Vercel

1. 打开浏览器访问：**https://vercel.com**
2. 点击 **"Sign Up"** 或 **"Login"**
3. 选择 **"Continue with GitHub"**
4. 授权 Vercel 访问您的 GitHub

### 步骤 2: 导入项目

1. 点击 **"Add New Project"**
2. 选择 **"Import Git Repository"**
3. 在列表中找到 `codemind-ai`
4. 点击 **"Import"**

### 步骤 3: 配置项目

在配置页面填写:

```
Framework Preset: Vite
Root Directory: frontend
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```

### 步骤 4: 开始部署

1. 点击 **"Deploy"**
2. 等待 2-3 分钟
3. 部署成功后会显示:
   - ✅ Ready
   - 🌐 访问链接：`https://codemind-ai-xxx.vercel.app`

### 步骤 5: 访问网站

在浏览器打开部署链接:
```
https://codemind-ai-xxx.vercel.app
```

## 🔧 方案 B: 完整部署 (前端 + 后端)

### 第一部分：部署后端到 Railway

#### 步骤 1: 访问 Railway

1. 打开：**https://railway.app**
2. 使用 GitHub 账号登录
3. 点击 **"New Project"**

#### 步骤 2: 创建后端服务

1. 选择 **"Deploy from GitHub repo"**
2. 选择 `codemind-ai` 仓库
3. 配置:
   ```
   Root Directory: backend
   Start Command: npm run dev
   ```

#### 步骤 3: 配置环境变量

在 Railway 面板添加:
```
DASHSCOPE_API_KEY=sk-your-api-key
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/api/v1
PORT=5000
NODE_ENV=production
CORS_ORIGIN=*
```

#### 步骤 4: 获取后端 URL

部署成功后获得:
```
https://your-app.railway.app
```

### 第二部分：修改前端配置

#### 修改 API 地址

编辑 `frontend/src/pages/MainPage.tsx`:

```typescript
// 将所有 http://localhost:5000 改为 Railway 的 URL
const response = await fetch('https://your-app.railway.app/api/analyze', {
  // ...
});
```

#### 重新推送代码

```bash
git add .
git commit -m 'deploy: 更新 API 地址为生产环境'
git push origin main
```

### 第三部分：重新部署前端到 Vercel

1. 在 Vercel 找到项目
2. 点击 **"Redeploy"**
3. 等待部署完成

## ⚡ 最快方法：使用 Vercel CLI

### 在您的电脑上执行:

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录 Vercel
vercel login

# 3. 进入项目目录
cd e:\trae\存项目\第二个代码优化项目\code-mind-ai

# 4. 部署前端
cd frontend
vercel

# 5. 生产部署
vercel --prod
```

### 按照提示操作:

1. `Set up and deploy?` → **Y**
2. `Which scope do you want to deploy to?` → 选择您的账号
3. `Link to existing project?` → **N** (首次部署)
4. `What's your project's name?` → **codemind-ai**
5. `In which directory is your code located?` → **./frontend**
6. `Want to override the settings?` → **N**

### 获得访问链接:

```
https://codemind-ai.vercel.app
```
