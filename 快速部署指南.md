## 🎯 最简单的部署方法（网页版）

### 步骤 1: 访问 Vercel

**打开浏览器，访问:**
```
https://vercel.com
```

### 步骤 2: 登录

1. 点击右上角的 **"Login"** 按钮
2. 选择 **"Continue with GitHub"**
3. 如果提示授权，点击 **"Authorize Vercel"**

### 步骤 3: 创建新项目

1. 点击 **"Add New Project"** 按钮
2. 点击 **"Import Git Repository"**

### 步骤 4: 选择您的项目

1. 在项目列表中找到 **`codemind-ai`**
2. 点击右边的 **"Import"** 按钮

### 步骤 5: 配置项目

在配置页面，填写以下信息:

```
Framework Preset:     Vite
Root Directory:       frontend
Build Command:        npm run build
Output Directory:     dist
Install Command:      npm install
```

### 步骤 6: 开始部署

1. 点击底部的 **"Deploy"** 按钮
2. 等待 2-3 分钟（可以看到部署进度）

### 步骤 7: 获得访问链接

部署成功后，您会看到:

```
🎉 Congratulations! Your project is deployed.

https://codemind-ai-xxx.vercel.app
```

**复制这个链接!** 这就是您项目的访问地址。
